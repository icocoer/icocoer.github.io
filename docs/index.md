# 工具库简介

欢迎使用 **ICOCOER 工具库**！这是一个为日常开发提速的实用工具集合。

## 它能做什么？

- 🛠 提供常用的工具函数，减少重复造轮子
- 📦 模块化设计，按需引入
- 🚀 零依赖，轻量快速
- 🔒 使用 TypeScript 编写，类型安全

## 特性一览

| 模块 | 说明 |
|------|------|
| `string` | 字符串处理工具 |
| `array` | 数组操作工具 |
| `date` | 日期格式化与计算 |
| `dom` | 常用 DOM 操作 |
| `http` | 请求封装 |
